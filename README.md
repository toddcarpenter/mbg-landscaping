# MBG Landscaping
# Using Outsourcify's HTML Starter with Webpack https://outsourcify.net

Push to github repo (changed from master to main):

```bash
git push origin main
```

Install the dependencies:

```bash
npm i
```

Development Build:

```bash
npm run build
```

Development Build:

```bash
npm run dist
```

Server start (open a new browser window):

```bash
npm run start
```
